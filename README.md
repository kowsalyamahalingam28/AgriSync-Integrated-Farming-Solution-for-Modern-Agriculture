# AgriSync — Smart Farming Assistant

AgriSync is a comprehensive, multilingual (Tamil + English) AI-powered smart farming web application. It is designed to assist farmers by providing intelligent insights and personalized recommendations.

## Features
- **Smart Farming Guide**: Educational content for agricultural best practices.
- **Disease Detection**: AI-powered analysis to identify and manage crop diseases.
- **Weather Forecasting**: Real-time accurate weather predictions tailored for farm planning.
- **Fertilizer & Crop Recommendations**: Suggests the optimal crops and fertilizers based on soil and climate.
- **Soil Health Scoring**: Tool for tracking and improving your farm's soil quality.
- **AI Crop Calendar**: Generates optimized planting and harvesting schedules.
- **Multilingual AI Chatbot**: Voice-enabled assistant that understands and responds in Tamil and English.

## Technologies Used
- React & Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

## Setup

1. Make sure you have Node.js installed.
2. Install the dependencies:
   ```sh
   npm install
   ```
3. Run the development server:
   ```sh
   npm run dev
   ```
