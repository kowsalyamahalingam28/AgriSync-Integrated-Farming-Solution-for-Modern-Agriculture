# AgriSync - Smart Farming Assistant

## Design System
- Earthy green primary: HSL 142 50% 28%
- Gold secondary: HSL 42 80% 50%
- Earth brown: HSL 30 40% 30%
- Leaf green: HSL 100 45% 40%
- Font: Poppins + Noto Sans Tamil
- Tamil/English bilingual (LanguageContext)
- App name: AgriSync (previously AgriSmart)

## Authors
MADHUMITHA A, AATHIKA N M, ARUNI B, KOWSALYA M

## Modules
1. Landing page (hero + stats + features + how it works + CTA + about)
2. Farming Guide (seasonal calendar + best practices + 14 crops)
3. Crop Detail pages (per crop with animated cards)
4. Disease Detection (image upload with step-by-step UI)
5. Weather Forecast (hero banner + responsive cards)
6. Fertilizer Recommendation (hero + NPK guide + tips)
7. Crop Recommendation (hero + form + animated results)
8. Farmer Profile (localStorage)
9. AI Chat Assistant (local responses + voice input)

## 14 Crops
Apple, Blueberry, Cherry, Corn, Grape, Orange, Peach, Pepper, Potato, Raspberry, Soybean, Squash, Strawberry, Tomato
